Making Images Look Good

A nice image with shadow and rounded corners

Here are some easy ways to improve your images:

  • Add border-radius for rounded corners
  • Use box-shadow for depth
  • Make sure they're responsive with max-width: 100%

Background Options

You can use these background types:

Example this is a gradient background!
Type CSS Code
Solid Color background: #f4f4f4;
Gradient background: linear-gradient(blue, white);
Image background: url('image.jpg');

Try This Background Code:

header {
    background: linear-gradient(135deg, #2c3e50, #34495e);
    color: white;
}
